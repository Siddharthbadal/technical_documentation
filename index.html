A software, in its most general sense, is a set of instructions and programms instructing a computer to do a sepcific task. Scripts, applications, programs are also used to describe software. A software program is a set of instruction or procedures, that are required to perform sectain type of operation.


      Software development is a magical art of creating a program which can perform a required task following a set of processes. Software development includes mainly three steps thinking, designing, and implementation. With software development, we must also know about Software development life cycle (SDLC). SDLC is a detailed plan explaining all major steps of a software development. There are mainly six steps in SDLC:

  • Planning

Basic termnology for sotware development

Here are few terms used in web development

Bugs: A bug is a program error.

Compiler: A program that translates a high level language program into an independent machine code program.

API: “Application Program Interface. How computers and applications communicate with one another.

Debugging: The process of locating and fixing or bypassing bugs(errors)

Attribute: Information about elements of a component in your website design/build.

Domain: The address for a website as entered into the browser. (www.freecodecamp.com!).

404: Error message when what was requested cannot be found; often because the link that was requested is either broken or dead.

Javascript


					script

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

       JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects


function firstCode(funName) { console.log(funName) } firstCode("JavaScript")

Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.

         Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server.


Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.
A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9). Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).


var x = 50; let y = 50;

Constants

You can create a read-only, named constant with the const keyword. The syntax of a constant identifier is the same as for a variable identifier: it must start with a letter, underscore or dollar sign and can contain alphabetic, numeric, or underscore characters.
A constant cannot change value through assignment or be re-declared while the script is running. It has to be initialized to a value.

const fcc= 3.3;

You cannot declare a constant with the same name as a function or variable in the same scope.

Understanding CSS

CSS stands for Cascading Style Sheets.

CSS desribes how HTML elements are to be displayed on the web page. CSS is a language for specifying how documents are presented to users how they are styled or laid out.

A Document is usally a text file structure using markup languages (HTML) and Presenting a documnet to a user means converting it into a form usable by your audience

p { color: blue; font-size; 20px } *p is for paragraph tag in HTML. color is blue, that to be applied on paragraph with font-size of 20 pixels.

Why use CSS

Three main advantages of using CSS:

  • Saves time: By using it multiple times for different web pages. and even inside a web page we do not have to create different style for different elements seperately.
  • Faster loading: It allows us to writeminimal code for HTML. Less code means less loading time.
  • Superior styles to HTML: CSS offers much wider array of attributes than HTML, so can give a better look to web page in comparison to HTML attributes.
  • Multiple Device Compatibility: Style sheets allow content to be optimized for more than one type of device. By using the same HTML document, different versions of a website can be presented.
